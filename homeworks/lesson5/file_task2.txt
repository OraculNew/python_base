4.2. Запуск агента сервера
4.2.1. Общая информация
Для того чтобы выполнить запуск кластера серверов «1С:Предприятия», следует
запустить агент сервера (ragent). Все дальнейшие действия будут выполнены системой
автоматически. При запуске агент сервера выполняет поиск списка кластеров,
зарегистрированных на данном компьютере.
Если список кластеров обнаружен, то агент сервера запускает указанные менеджеры
кластеров. С их помощью он получает информацию о рабочих процессах, которые
должны быть запущены в каждом из кластеров, и выполняет их запуск самостоятельно
или с помощью агентов других рабочих серверов кластера.
Если список кластеров не обнаружен, агент сервера создает кластер по умолчанию.
Кластер по умолчанию имеет следующие характеристики:

● номер сетевого порта - 1541;
● диапазон сетевых портов - 1560:1591;
● при запуске в отладочном режиме с использованием протокола HTTP, сервер
отладки использует порт 1550;
● один рабочий процесс, номер порта устанавливается из указанного диапазона.
